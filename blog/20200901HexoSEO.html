Hexo搜索引擎优化[SEO]

发表2020-09-01更新2020-09-10字数1.4k预计阅读时长7分阅读次数
hexo

应 白小羊 请求，我撰写了本文。

本文简单介绍一些重要的 SEO 操作。它们经过了本站实践的检验。

但您请注意，一切 SEO ，都是为了让搜索引擎高效收录您的网页（那当然）。所以搜索引擎们的官方说明才是最重要的，而且它们最完整。如果您有时间，按照本文优化完毕后，请研读官方说明。国际看谷歌，内地看百度。快速传送： [转]搜索引擎优化[SEO]新手官方指南

url

实际上，url 对网站权重影响很大。

https

一定要启用全站 https 。

域名

强烈建议使用自己的域名。github.io（已被 DNS 污染，且进入工信部黑名单）,gitee.io等随时可能被墙，有自己的域名可以方便地更换服务商。确实没闲钱的做题家们可以使用 免费子域名

关于购买域名，我的建议：计划备案用.cn，便宜；不计划备案用.com等内地支持备案的域名。且尽量不使用中文域名。这是综合 SEO 效果、域名价格、域名政策、访客熟悉度等的结果。还有，万一您有一天想备案呢？

内地搜索引擎偏好备案的网站，收录一般更快。毕竟方便追责，网站更“合法”。

我没有备案的计划。域名注册、DNS 解析、服务器全走的国际服务。

资源地址

请降低文章地址的层级。hexo 文章默认的资源地址采用多级时间分类存放（https://xxx.github.io/2020/04/07/xxx/），如此层级不利于搜索引擎爬取。将 hexo 的 _config.yml 中的属性修改为

就可以得到文章 URL 如：https://cjh0613.com/20200603HexoSubmitUrlsToSearchEngine.html

更改后，最好能兼容一下旧版链接。推荐在 404.html 写 js 跳转。

另外，部分搜索引擎还会从 URL 获取信息。如，上面这个链接被收录后，百度记录为是在 20200603 发布的。实际上，我是在 2020-07-06 发布的这篇文章，且网页中没有与六月三日相关的文字。

向搜索引擎提交

主动提交新链接

我开发的：主动推送Hexo博客新链接至谷歌、必应、百度搜索引擎站长平台以提升网站收录质量和速度的插件hexo-submit-urls-to-search-engine完美实现这操作。

sitemap

用插件生成针对谷歌、百度的 sitemap 并提交。在 hexo 官方网站插件大全页可以搜到很多生成 sitemap 的插件。一定要使用搜索引擎站长平台的抓取测试工具。抓取站点地图，确保能 100% 成功抓取，如不能，请排查是服务器原因还是搜索引擎的问题。是服务器的原因，请尽早更换，否则会因此被降权。

html标签

首先，请选用成熟的 Hexo 主题。它们会自动生成对 SEO 比较重要的 Html 标签。如果您的主题不会自动生成这些标签，请自行修改主题源码以支持。这里就不展开讲了，您可自行在搜索引擎们的官方说明使用页内搜索找到详细配置说明。

这些重要的标签设置有：

  • rel=“canonical”，选定的规范网页
  • nofollow ，避免爬取站外链接传递权重，可用 hexo 插件实现，请不要给友情链接加此属性
  • img alt ,图片说明
  • Meta description，便于搜索引擎智能摘要
  • author
  • property=“article:……”，便于搜索引擎智能摘要
  • property=“og:……”，便于搜索引擎智能摘要
  • ……

标题

百度偏好的标题字符使用：

服务器

请选用稳定的服务器。

请选用对搜索引擎友好的服务器托管您的站点。请注意，对搜索引擎友好对访问速度要求不太高。验证是否对搜索引擎友好的常用方法：使用搜索引擎站长平台的抓取测试功能。抓取网页，robots.txt，站点地图等。能 100% 成功抓取即可。

实测 Gitee Page 屏蔽了访问 robots.txt，非常……不太友好

robots.txt

robots.txt是一种存放于网站根目录下的ASCII编码的文本文件，它通常告诉网络搜索引擎的漫游器（又称网络蜘蛛），此网站中的哪些内容是不应被搜索引擎的漫游器获取的，哪些是可以被漫游器获取的。

搜索引擎一定会尝试访问此文件。如果没有此文件，搜索引擎会等待至超时，有此文件一定比没有更好。您在 hexo source 文件夹下新建robots.txt并按规则编辑即可。

引流

1、在知乎、CSDN、简书等权重较高的网站转载文章，并注明原文来自您的站点，放个链接。

像微信公众号、头条号等搜索引擎都被拒的局域网站就不用管了。

2、友情链接。

欢迎交友交流，交换本站友情链接请点这里

坚持更新

内容是核心。

坚持更新这一点非常重要。“我的一篇文章刚发布5分钟，即可在必应搜索到”的那段日子，我每天至少新发文一篇。

当然，还是请不要在个人站上堆砌低质量文章。
