修改 DNS 服务器

它可以解决电信运营商 DNS 污染（域名指往不正确的IP地址），造成的部分网站（如 Gist ，蓝奏云网盘，github.io 博客等）无法打开的问题。【若干年前，甚至可以通过设置 DNS 打开被墙的 **百科、*歌】

建议修改 DNS 服务器为：

首选DNS服务器 114.114.114.114

备用DNS服务器 8.8.8.8

具体设置方法请继续阅读。

win10

  1. 在任务栏右下角"网络"上右键单击
  2. 在"打开网络和 Internet设置" 上左键单击
  3. 在"更改适配器选项"上左击（在"设置"窗口中）
  4. 在"以太网"（本地连接）上右键单击（在“网络连接"中）
  5. 在“属性”上左键单击
  6. 双击“Internet协议版本4（TCP/PV4）”
  7. 选择"使用下面的DNS服务器地址"
  8. 修改为上述 DNS 服务器

安卓

在无 Root 时：

连接 wifi 后，可进入手机 wifi 配置，在该 wifi 的设置界面设置 DNS 服务器。

还可使用 Hosts Go、Nebulo（支持DoH，DoT）等软件，通过虚拟专用网络使用选定的 DNS 。

当然，最好能启用 DOT/DOH ,因为：

传统的 DNS 查询和应答采用 UDP 和 TCP 明文传输，存在网络监听、DNS 劫持、中间设备干扰等风险：

  • 网络监听风险：即便用户采用 HTTPs 加密的方式访问站点，DNS 查询应答并没有采用加密传输
  • DNS 劫持：传统 DNS 应答数据会被篡改，用户的访问会被路由到钓鱼网站和恶意站点
  • 中间设备干扰：主要是一些防火墙对 DNS 查询的干扰，基于域名的过滤，还有大包 MTU 分片的影响等

DoH 、DoT 能较好地避免上述风险。PC 上火狐，谷歌浏览器均已支持。具体请自行 *歌/百度
